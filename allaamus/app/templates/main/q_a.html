Sophos UTM maili gondermeyi reddediyor

Merhaba Sophos UTM cihazi on-premise exchange server uzerinden gonderilen BCC maillerini kullanicilara iletmiyor. O mail icindeki kimseye gitmiyor mail to, cc ya da bcc olduguna bakmaksizin. Eger ki ben bcc'ye bir alici adresi eklediysem. "internal server error" diyip reddediyor ama bcc'den ismi temizler de ayni maili gondermeye calisirsam mail gidiyor. Sophos loglarinda her hangi bir kayit da yok bu reddetmeye dair. Boyle bir sorunla karsilasan oldu mu acaba?

PLESK Web Sitelerinin Aktarımı

Selamlar, Plesk üzerinde tuttugumuz web sitelerinin farkli ip li server a tasimak istiyoruz. Bu islem icin psa bacuplarini aldik ve yeni server üzerinde actik. Ancak tum domainlerin dns ayarlarini tekrar degistirmemek icin domainlerin yonlendirmesini eski plesk server uzerinden yeni plesk e dogru yapmak isityoruz. Bu islem isin Domainlerin DNS Setting bolumlerinden PTR degerlerini yeni server in ip degeriyle degistirdik, fakat bir sonuc alamadik. Bu islemi nasil gerceklestiririz. Tesekkurler
